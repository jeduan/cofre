# Cofre

Cofre is a highly configurable escrow to allow safe peer-to-peer trades.

## Features

- Types of trades and examples
  - SPL - SPL
    - NFT to NFT swap
  - SPL - SOL
    - eg: Selling an NFT for 1 SOL
  - SOL - SPL
    - eg: Offering to buy 1000 SBR tokens for 1 SOL
